Web Scraper API Documentation

Learn how to extract structured data from websites using our API

API Endpoint

The API has a single endpoint that accepts POST requests:

POST /scrape

Request Format

All requests should be sent with the following JSON structure:

{
  "url": "string",      // The website URL to scrape
  "query": "string",    // Natural language description of what to find
  "maxSteps": number,   // Maximum navigation steps (1-10)
  "targetSchema": {     // JSON Schema of the desired data
    "type": "object",
    "properties": {...},
    "required": [...]
  }
}
Field Type Description
url string The target website URL to scrape
query string Natural language description of what information to extract
maxSteps number Maximum number of navigation steps (1-10)
targetSchema object JSON Schema describing the structure of the data to extract

Example Schemas

Simple Email Extraction
{
  "url": "https://maravian.com",
  "query": "find the email from the site",
  "maxSteps": 5,
  "targetSchema": {
    "type": "object",
    "properties": {
      "email": { "type": "string" }
    },
    "required": ["email"]
  }
}
Contact Information
{
  "url": "https://example.com",
  "query": "Find the contact information of the main person, including their name, position, email, and phone if available",
  "maxSteps": 5,
  "targetSchema": {
    "type": "object",
    "properties": {
      "name": { "type": "string" },
      "position": { "type": "string" },
      "email": { "type": "string" },
      "phone": { "type": "string" }
    },
    "required": ["name", "email"]
  }
}
Project List
{
  "url": "https://example.com",
  "query": "Find all projects listed on the site, including their titles, descriptions, and technologies used",
  "maxSteps": 5,
  "targetSchema": {
    "type": "object",
    "properties": {
      "projects": {
        "type": "array",
        "items": {
          "type": "object",
          "properties": {
            "title": { "type": "string" },
            "description": { "type": "string" },
            "technologies": {
              "type": "array",
              "items": { "type": "string" }
            }
          },
          "required": ["title", "description"]
        }
      }
    },
    "required": ["projects"]
  }
}

Response Format

The API returns JSON responses in the following format:

{
  "isError": boolean,    // true if an error occurred
  "message": string,     // success/error message
  "data": object | null  // extracted data matching the targetSchema
}
